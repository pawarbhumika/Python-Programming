# Python Programming

A collection of Python learning materials and practice code, organized by topic. This repository is intended as a personal/learning reference covering core Python fundamentals along with the NumPy and Pandas libraries commonly used for data analysis.

## 📂 Repository Structure

```
PythonProgramming/
├── Fundamentals/   # Core Python concepts and exercises
├── Numpy/          # NumPy library examples and practice
└── Pandas/         # Pandas library examples and practice
```

### Fundamentals
Scripts and notes covering the building blocks of Python — syntax, data types, control flow, functions, data structures, and other core language concepts.

### Numpy
Examples demonstrating array creation, indexing, broadcasting, mathematical operations, and other features of the [NumPy](https://numpy.org/) library.

### Pandas
Examples demonstrating DataFrames, Series, data cleaning, filtering, grouping, and other features of the [Pandas](https://pandas.pydata.org/) library.

## 🛠️ Getting Started

### Prerequisites
- Python 3.x
- pip

### Installation
Clone the repository:
```bash
git clone https://github.com/deveshdubey18/PythonProgramming.git
cd PythonProgramming
```

Install the required libraries (for the Numpy/Pandas sections):
```bash
pip install numpy pandas
```

### Usage
Navigate into any folder and run the scripts individually with Python:
```bash
python Fundamentals/<script_name>.py
```
